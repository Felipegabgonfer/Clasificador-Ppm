# Clasificador-Ppm

Este proyecto toma las imágenes ppm de un directorio que se tiene que pasar mediante línea de comandos
y clasifica sus píxeles dependiendo de si son más azules, más rojos, o más verdes. Estos píxeles son posteriormente
copiados en 3 imágenes nuevas, una solo con los píxeles azules, otra con solo píxeles verdes y otra con solo píxeles rojos,
teniendo los píxeles de los otros dos colores en negro. Los píxeles que no tengan un color predominante se pintarán en negro en las tres imágenes.
